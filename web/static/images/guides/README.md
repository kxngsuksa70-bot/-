# วิธีเพิ่มรูปคู่มือการใช้งาน

## ตำแหน่งไฟล์
ใส่ไฟล์รูปภาพ PNG ที่:
```
c:\Users\Nanoatom\Downloads\puyfai\web\static\images\guides\student-guide.png
```

## ขนาดแนะนำ
- ความกว้าง: 800-1200 px
- ความสูง: ตามสัดส่วน
- รูปแบบ: PNG (รองรับความโปร่งใส)

## ฟีเจอร์
- แสดงเฉพาะเมื่อนักศึกษา login (ไม่แสดงให้อาจารย์)
- มี checkbox "ไม่ต้องแสดงอีก"
- ถ้าติ๊กถูก จะไม่แสดงในครั้งต่อไป
- สามารถ reset โดยลบ localStorage: `localStorage.removeItem('hideStudentGuide')`

## ตัวอย่างรูปที่ควรมี
1. คู่มือหน้าหลัก
2. วิธีค้นหาอาจารย์
3. วิธีดูตารางสอน
4. วิธีใช้งานต่างๆ

## วิธีทดสอบ
1. Login ด้วย student1/1234
2. Popup จะขึ้นทันที
3. Click "✕ ปิด" หรือ click นอก popup เพื่อปิด
4. ติ๊ก "ไม่ต้องแสดงอีก" ถ้าไม่ต้องการเห็นอีก
